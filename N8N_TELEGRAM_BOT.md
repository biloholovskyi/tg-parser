# Создание Telegram бота для n8n

## Шаг 1: Создай бота через @BotFather

1. Открой Telegram и найди [@BotFather](https://t.me/BotFather)
2. Отправь команду `/newbot`
3. Введи имя бота (например: `My Channel Parser`)
4. Введи username бота (например: `my_channel_parser_bot`)
5. BotFather вернет токен в формате:

```
1234567890:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
```

**⚠️ СОХРАНИ ЭТОТ ТОКЕН!** Он понадобится для n8n.

## Шаг 2: Настрой бота

Отправь BotFather команды для настройки:

### Установи описание

```
/setdescription
```

Выбери своего бота и введи описание:
```
Бот для получения постов из Telegram каналов
```

### Установи команды

```
/setcommands
```

Выбери своего бота и введи команды:
```
getposts - Получить посты из канала
help - Помощь
```

## Шаг 3: Проверь бота

1. Найди своего бота в Telegram по username
2. Нажми **Start**
3. Отправь `/help` - бот пока не ответит (это нормально)

## Шаг 4: Получи Chat ID (для отправки сообщений)

Если хочешь, чтобы бот отправлял сообщения в определенный чат:

1. Добавь бота в нужный чат или группу
2. Используй этот API запрос для получения Chat ID:

```
https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
```

Открой эту ссылку в браузере и найди `"chat":{"id":...}` - это твой Chat ID.

## Шаг 5: Подключи бота к n8n

Теперь используй Bot Token в n8n workflow. См. [N8N_WORKFLOW_GUIDE.md](./N8N_WORKFLOW_GUIDE.md).

## Команды бота

После настройки n8n, бот будет принимать такие команды:

### `/getposts <channel> <hours>`

Получает посты из канала за указанное количество часов.

**Примеры:**

```
/getposts durov 24
```
Получить посты из @durov за последние 24 часа.

```
/getposts breakingmash 48
```
Получить посты из @breakingmash за последние 48 часов.

### Формат ответа

Бот вернет:
- Краткое саммари постов (сгенерированное ИИ)
- Ссылки на оригинальные посты
- Количество обработанных постов

## Безопасность

**⚠️ Важные моменты:**

1. **Не делись Bot Token ни с кем!** С ним любой может управлять твоим ботом.
2. Если токен утек - сгенерируй новый через BotFather командой `/revoke`
3. Рекомендуется ограничить доступ к боту только для определенных пользователей (можно реализовать в n8n)

## Продвинутые настройки

### Ограничение доступа к боту

В n8n можно добавить проверку User ID:

```javascript
// В Function Node перед обработкой
const allowedUsers = [123456789, 987654321]; // Твои Telegram User IDs
const userId = $input.item.json.message.from.id;

if (!allowedUsers.includes(userId)) {
  return [{
    json: {
      reply: "У вас нет доступа к этому боту."
    }
  }];
}

// Продолжить обработку
return $input.all();
```

### Получить свой User ID

Отправь любое сообщение боту [@userinfobot](https://t.me/userinfobot) - он вернет твой User ID.
